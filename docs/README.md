# Documentation index

Docs are split by audience:

| | |
|--|--|
| **[`docs/user/`](user/README.md)** | People who **use the `tray` CLI** (install, daily commands, hooks, output formats). |
| **[`skills/`](skills/README.md)** | **Agent skills** for Claude/Cursor (operating `tray`); canonical [`skills/tray-cli/SKILL.md`](skills/tray-cli/SKILL.md). |
| **[`docs/maintainers/`](maintainers/README.md)** | People who **work on this repository** (tests, releases, distribution, CI). |

The [root `README.md`](../README.md) is written primarily for **CLI users**; **maintainers and contributors** are directed to [`docs/maintainers/README.md`](maintainers/README.md) from the bottom of that file.
